## Where applications meet the network

The Application layer is the one users actually touch — not the application itself, but the
**protocols** applications speak to exchange data over the network: **HTTP** for the web, **DNS** to
turn names into addresses, **SMTP/IMAP** for email, **SSH** for secure shells, **DHCP** to obtain an
address. It defines message formats and the rules of exchange for concrete services.
